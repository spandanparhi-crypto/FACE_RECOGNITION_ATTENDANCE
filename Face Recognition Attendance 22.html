An automated, contactless, and highly accurate solution to manage class and workspace attendance effortlessly using advanced Computer Vision techniques.

About the Project

The Face Recognition Attendance System leverages modern biometric technologies to identify and verify a person dynamically through an active camera stream. By processing video frames against a secure database of registered face templates, the system automatically logs attendance in real-time.

Traditional methods like roll calls or physical logbooks are prone to errors and proxy signatures. This system eliminates human overhead, reduces proxy attendance, saves valuable class time, and generates seamless digital audit reports.

Project Objectives

  • Automate attendance tracking system.
  • Drastically improve tracking data accuracy.
  • Minimise manual data entry and human error.
  • Generate real-time comprehensive attendance reports.
  • Ensure security and integrity of student records.

System Workflow

1

Register Student: Provide details such as Name, Roll Number, and Class details.

2

Capture Face Images: The camera snaps multiple samples of the student's face from various angles.

3

Train Face Model: Deep learning embeddings process the gathered imagery mapping facial keypoints.

4

Detect Face (Live): Active frame readers identify facial geometries dynamically through cameras during check-in.

5

Database Match: Facial hashes map immediately onto saved templates looking for valid validation profiles.

6

Mark Attendance: The match system registers validation metrics upon positive identification hits.

7

Store Log: Time, date, tracking numbers drop clean timestamp markers onto tables.

8

Generate Reports: Admins export customized files directly sorted by specific metric categories.
